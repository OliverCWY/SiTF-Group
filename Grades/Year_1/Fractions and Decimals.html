Fractions and Decimals

  • A fraction is a number that can be written as `a/b`, `b != 0`.
  • A decimal is a number written using a decimal point.
  • A terminating decimal is a decimal number which has digits that terminate(do not go on forever) For example, `1/5 = 0.2`.
    On the other hand, a decimal which does not terminate is called a non-terminating decimal. For example, `1/3` and `1/7` are non-terminating decimals.
  • If the digits after a decimal point in a non-terminating decimal repeat themselves in a pattern, it is called a recurring decimal, if they do not repeat in a pattern, it is called a non-recurring decimal.
Learning Objectives
  • Identify proper and improper fractions, equivalent fractions, unit fractions, terminating decimals, non terminating(recurring) decimals.
  • Evaluate numerical expressions involving fractions and decimals.
  • Predict if a fraction is equivalent to a recurring or terminating decimal.
  • Convert a recurring decimal to a fraction and vice versa.
